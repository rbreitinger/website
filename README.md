# Yorokobi Games

A small collection of browser games — made by one person, offered completely free.

No ads. No analytics. No cookies. No tracking scripts of any kind.  
Every game runs in your browser without installation, on any device, at any age.  
Nothing on this site contains violence, explicit content, or anything you wouldn't 
want a child to see. That's not a policy buried somewhere — it's the whole point.

**The player is the guest — not the product.**

🌐 [yorokobi-games.com](https://yorokobi-games.com)

---

## About This Repository

This is the full source code of the Yorokobi Games website — published openly for
transparency, learning, and community contribution.

There is no framework, no build system, no dependencies.  
Plain HTML, CSS and JavaScript. Runs anywhere, on anything.

---

## Structure
```
index.html          → main catalogue / landing page
game.html           → legacy game shell (v1)
game-v2.html        → improved game shell (v2)
game-phone.html     → mobile-first phone shell (current)
webgames/           → individual game folders
assets/             → shared images and resources
```
